Застосунок TWEETS

При виконанні данного завдання використовувалось:
    - VITE;
    - GIT HUB
    - MOCKAPI.io
    - JS
    - REACT
    - AXIOS
    - REACT LOADER SPINNER
    - REACT ROUT
    - CSS.MODULE
    - бібліотека SWEETALERT
    - local storage

Застосунок має маршрутизацію. Перша сторінка домашня, друга для відображення юзерів. При кліці на кнопку FOLLOW змінюється колір кнопки, додається один фоловер та відбувається запит на бекенд, де у поточного юзера перезаписується кількість фоловерів. При кліці на кнопку FOLLOWING відбувається відписка від юзера і логіка відбувається в зворотньому порядку. При перезавантаженні сторінки дії користувача зберігаються. 
Якщо в адресне поле введено не коректну адресу, користувач потрапляє на сторінку NOT FOUND, де отримує сповіщення про не коректну адрессу і отримує доступ до посиляння для переходу на головну сторінку.
